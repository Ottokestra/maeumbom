# 마음봄 (Maeumbom) 서비스 소개

> **감정 교감 인터페이스** - AI 기반 감정 케어 및 관계 연습 서비스

마음봄은 사용자의 감정을 이해하고 공감하는 AI 파트너 "봄이"와 함께 감정을 관리하고, 관계를 연습하며, 마음의 건강을 돌보는 종합 감정 케어 플랫폼입니다.

---

## 📱 핵심 서비스

### 1. 🏠 홈 (Home)
**위치:** `/app/home`

일일 감정 체크 및 주요 기능 대시보드

**주요 기능:**
- **일일 감정 체크**: 17개 감정 캐릭터 중 오늘의 기분 선택
- **감정 기반 UI**: 선택한 감정에 따라 배경색과 캐릭터가 동적으로 변경
  - 좋음 (good): 노란색 배경 (#FFB84C)
  - 보통 (neutral): 초록색 배경 (#63C96B)
  - 나쁨 (bad): 파란색 배경 (#6C8CD5)
- **2x3 그리드 대시보드**: 
  - 감정 캐릭터 표시
  - 5개 주요 기능 바로가기 (봄이와 대화, 기억서랍, 마음리포트, 마음연습실, 신조어퀴즈)
- **기분 체크 리마인더**: 미체크 시 자동 팝업

**화면:**
- `home_new_screen.dart` - 메인 홈 화면
- `daily_mood_check_screen.dart` - 감정 선택 화면

### 2. 💬 봄이와 대화 (Bomi Chat)
**위치:** `/app/chat`

AI 파트너 "봄이"와의 감정 교감 대화

**주요 기능:**
- **음성 대화**: 마이크를 통한 자연스러운 음성 대화
  - 실시간 음성 인식 (STT)
  - AI 음성 응답 (TTS)
  - 음성 파동 시각화
- **텍스트 대화**: 키보드 입력을 통한 채팅
- **감정 분석**: 대화 내용 기반 감정 상태 파악
- **캐릭터 애니메이션**: 대화 중 감정에 맞는 Lottie 애니메이션
- **선택지 응답**: 상황에 맞는 선택지 제공
- **대화 모드 전환**: 음성 ↔ 텍스트 모드 자유 전환

**화면:**
- `bomi_screen.dart` - AI 봄이 대화 화면
- `chat_screen.dart` - 일반 채팅 화면
- `chat_list_screen.dart` - 대화 목록 화면

**기술 스택:**
- 음성 인식/합성 (STT/TTS)
- WebSocket 실시간 통신
- Lottie 애니메이션

---

### 3. 🗂️ 기억 서랍 (Alarm/Memory)
**위치:** `/app/alarm`

일정, 알림, 기억을 관리하는 똑똑한 서랍

**주요 기능:**
- **일일 이벤트 분석**: `/analyze-daily` API를 통한 날짜별 이벤트 요약
- **날짜 범위 선택**: 이전/다음 날짜 네비게이션
- **3가지 타입 관리**:
  - 🔔 **알림 (Alarm)**: 시간 기반 알림 (토글 on/off)
  - ❤️ **기억 (Memory)**: 소중한 기억 저장
  - 📅 **이벤트 (Event)**: 일정 관리
- **키워드 기반 태그 시스템**:
  - `#알람/알림` → 노랑/오렌지
  - `#중요/긴급` → 붉은색
  - `#약속/미팅/회의` → 초록색
  - `#기억/추억` → 핑크/산호색
  - `#이벤트/행사` → 파란색
- **스와이프 삭제**: Dismissible 제스처로 간편 삭제
- **2줄 레이아웃**: 날짜/타입/시간 + 아이콘/내용/태그

**화면:**
- `alarm_screen.dart` - 알람 메인 화면
- `memory_list_screen.dart` - 기억 목록 화면

### 4. 📊 마음리포트 (Report)
**위치:** `/app/report`

주간 감정 분석 및 통계 리포트

**주요 기능:**
- **주간 감정 온도**: Fear & Greed Index 스타일 게이지
- **요일별 감정 캐릭터**: 7일간의 감정 변화 시각화
- **감정 분석 상세**: 주간 감정 패턴 및 인사이트
- **주차별 네비게이션**: 이전/다음 주 리포트 조회
- **3페이지 구성**:
  - Page 1: 이번주 감정 온도 (반원형 게이지)
  - Page 2: 요일별 감정 캐릭터 (7일 타임라인)
  - Page 3: 이번주 감정 분석 상세 (통계 및 인사이트)

**화면:**
- `report_screen.dart` - 리포트 메인 화면
- `weekly_emotion_report_screen.dart` - 주간 감정 리포트
- `pages/report_page1.dart` - 감정 온도 페이지
- `pages/report_page2.dart` - 요일별 감정 페이지
- `pages/report_page3.dart` - 감정 분석 상세 페이지

## 🎨 디자인 철학

### 감정 교감 인터페이스
마음봄은 단순한 앱이 아니라 **감정을 교감하는 인터페이스**입니다.

#### 핵심 원칙
1. **캐릭터 중심 (Character-First)**: 17개 감정 캐릭터를 통한 상호작용
2. **감정 중심 (Emotion-First)**: 감정 상태에 따른 동적 UI
3. **음성 중심 (Voice-First)**: 자연스러운 음성 대화 우선
4. **화이트 스페이스**: 충분한 여백으로 시각적 안정감
5. **직관적 인터랙션**: 최소한의 버튼, 최대한의 공감

### 17개 감정 캐릭터 시스템

#### 긍정 감정 (7개)
- 기쁨 (joy) - 해바라기 🌻
- 흥분 (excitement) - 별 ⭐
- 자신감 (confidence) - 사자 🦁
- 사랑 (love) - 펭귄 🐧
- 안심 (relief) - 사슴 🦌
- 깨달음 (enlightenment) - 전구 💡
- 흥미 (interest) - 부엉이 🦉

#### 부정 감정 (10개)
- 불만 (discontent) - 당근 🥕
- 수치 (shame) - 복숭아 🍑
- 슬픔 (sadness) - 고래 🐋
- 죄책감 (guilt) - 곰 🐻
- 우울 (depression) - 돌 🪨
- 무료 (boredom) - 나무늘보 🦥
- 경멸 (contempt) - 가지 🍆
- 화 (anger) - 불 🔥
- 공포 (fear) - 쥐 🐭
- 혼란 (confusion) - 로봇 🤖

## 🛠 기술 스택

### Frontend
- **Framework**: Flutter 3.x
- **상태관리**: Riverpod
- **로컬 DB**: Drift (SQLite)
- **애니메이션**: Lottie
- **음성**: STT/TTS
- **네트워킹**: Dio, WebSocket

### Backend Integration
- **API**: RESTful API
- **실시간 통신**: WebSocket
- **인증**: OAuth 2.0 (Google, Kakao, Naver)
- **알림**: Firebase Cloud Messaging

## 📊 주요 API 엔드포인트

### 채팅
- `POST /chat/send` - 메시지 전송
- `GET /chat/history` - 대화 기록 조회
- `WebSocket /chat/stream` - 실시간 대화

### 감정 분석
- `POST /analyze-daily` - 일일 이벤트 분석
- `GET /emotion/weekly` - 주간 감정 통계

### 알람/기억
- `GET /alarms` - 알람 목록 조회
- `POST /alarms` - 알람 생성
- `PUT /alarms/{id}` - 알람 수정
- `DELETE /alarms/{id}` - 알람 삭제

### 트레이닝
- `POST /training/scenario/generate` - AI 시나리오 생성
- `POST /training/quiz/start` - 퀴즈 시작
- `POST /training/quiz/submit` - 퀴즈 제출

## 🎯 타겟 사용자

### 주요 타겟
- **갱년기 여성**: 감정 변화가 큰 시기의 감정 관리
- **감정 표현이 어려운 사람**: AI와의 안전한 대화 연습

3. **저녁**: 기억 서랍에 하루 기록
4. **주말**: 마음리포트로 주간 감정 돌아보기
5. **여유 시간**: 마음연습실에서 관계 연습

## 📈 서비스 특징

### 차별화 포인트
1. **17개 감정 캐릭터**: 세밀한 감정 표현 및 시각화
2. **음성 우선 인터페이스**: 자연스러운 대화 경험
3. **AI 시나리오 생성**: 개인 맞춤형 관계 연습
4. **키워드 기반 태그**: 직관적인 기억 분류
5. **감정 기반 동적 UI**: 사용자 감정에 반응하는 인터페이스

### 핵심 가치
- **공감 (Empathy)**: AI가 사용자의 감정을 이해하고 공감
- **성장 (Growth)**: 관계 연습을 통한 소통 능력 향상
- **기록 (Record)**: 감정과 기억의 체계적 관리
- **분석 (Analysis)**: 데이터 기반 감정 패턴 인사이트
